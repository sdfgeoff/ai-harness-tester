Create an interactive geodesic chess game as a web application using npm, typescript and threejs. When done, build it so there is a `dist` folder containing `index.html`.

Geodesic Chess is a chess variant played on a spherical grid of hexagons and pentagons (a GeoBoard). Read the rules in `rules/RULES.md` for the complete specification including board layout, piece movement, and game mechanics.

The application should:

- Render the geodesic board (hexagons and pentagons) in 3D, allowing the user to click/drag around it, select pieces etc.
- Show all pieces with their starting positions as 3d objects.
- Allow the user to move pieces only according to the rules.
- Handle captures, check, and checkmate.
