Create an interactive geodesic chess game as a web application using npm, react, typescript and vite. When done, build it so there is a `dist` folder containing `index.html`.

Geodesic Chess is a chess variant played on a spherical grid of hexagons and pentagons (a GeoBoard). Read the rules in `rules/RULES.md` for the complete specification including board layout, piece movement, and game mechanics.

The application should:
- Render the geodesic board (hexagons and pentagons)
- Show all pieces with their starting positions
- Support piece movement according to the rules
- Handle captures, check, and checkmate
- Display the current game state clearly
